# Future_ML_03
Resume Screening System

Built an ML system that reads resumes and automatically scores, ranks and screens candidates based on a given job role.

Trained on 2,484 real resumes across 24 job categories. The system matches each resume against a job description, gives a match score, assigns a grade and flags any missing skills.

Tools: Python, Pandas, NLTK, Scikit-learn, plotly.graph_objects, streamlit

Main Attachments:
- resume_screening.ipynb - full code and results
- candidate_rankings.png - top 10 candidate ranking chart
- score_distribution.png - match score distribution chart
- visual3_app.py- full code for visual interface

Dataset: Resume Dataset from Kaggle
